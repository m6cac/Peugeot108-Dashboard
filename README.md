# Peugeot108-Dashboard

A custom open-source digital dashboard for the Peugeot 108.

## Project Goals

This project aims to build a standalone dashboard using an ESP32 that communicates with the vehicle through the OBD-II port.

### Planned Features

- Engine RPM
- Vehicle Speed
- Coolant Temperature
- Battery Voltage
- Fuel Level
- Shift Light
- Diagnostic Trouble Codes (DTCs)
- TFT Colour Display
- Custom PCB
- 3D Printed Enclosure

## Hardware

- ESP32
- TFT Display
- OBD-II Interface
- Custom PCB
